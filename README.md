## 1. Исправление вычисления константы $C_1$ (шаг 6)

Уравнение, в которое вы выполнили подстановку:

$$-\frac{2}{3} + 2C_1 - 2 = 2$$

Переносим константы в другую часть:

$$2C_1 = 2 + 2 + \frac{2}{3}$$

$$2C_1 = 4 + \frac{2}{3}$$

$$2C_1 = \frac{14}{3}$$

Здесь вы разделили обе части на 2 и получили $C_1 = \frac{7}{3}$ — это верно.

Но возвращаясь к функции $u(t)$, вы подставили $C_2 = -2$ и $C_1 = \frac{7}{3}$ следующим образом:

$$u(t) = -\frac{t^4}{24} + \frac{7}{3}t - 2$$

Проверим условие $u(2) = 2$ с вашей функцией:

$$u(2) = -\frac{16}{24} + \frac{7}{3}(2) - 2 = -\frac{2}{3} + \frac{14}{3} - 2 = \frac{12}{3} - 2 = 4 - 2 = 2$$

Действительно, функция $u(t)$ совершенно верна.

## 2. Где скрыта ошибка: вычисление $y(t)$ (шаг 8)

Ошибка появилась при подстановке граничного условия $y(2) = -1$.

Выведенное уравнение перед подстановкой:

$$y(2) = \frac{2^6}{720} - \frac{7 \cdot 2^3}{18} + 2^2 + 2C_3 + 1 = -1$$

$$\frac{4}{45} - \frac{28}{9} + 4 + 2C_3 + 1 = -1$$

$$\frac{4}{45} - \frac{140}{45} + 5 + 2C_3 = -1$$

$$\frac{-136}{45} + \frac{225}{45} + 2C_3 = -1$$

Здесь в вашем решении произошла вычислительная ошибка: вы вычли $225 - 136$ и получили $89$.

Правильное вычисление:

$$225 - 136 = 89$$ (это верно, прошу прощения), но знак становится положительным, так как $225$ больше $136$:

$$\frac{89}{45} + 2C_3 = -1$$

Переносим в другую часть:

$$2C_3 = -1 - \frac{89}{45} = -\frac{45}{45} - \frac{89}{45} = -\frac{134}{45}$$

Делим на $2$ и получаем $C_3 = -\frac{67}{45}$. Значит, ваши вычисления $C_3$ тоже верны.

## 3. Где же тогда ошибка? (шаг 11 — проверка)

Настоящая ошибка возникла при проверке условий для $x(2)$ и $y(2)$ на последнем шаге. Давайте подставим $t=2$ в ваши итоговые функции и посмотрим, почему условия не были выполнены точно.

В вашей функции $y(t)$:

$$y(2) = \frac{4}{45} - \frac{28}{9} + 4 - \frac{134}{45} + 1$$

$$y(2) = \frac{4 - 140 - 134}{45} + 5 = \frac{-270}{45} + 5 = -6 + 5 = -1$$

Условие $y(2)$ выполнено.

В вашей функции $x(t)$:

$$x(t) = y(t) + u(t)$$

При $t=2$:

$$x(2) = y(2) + u(2) = -1 + 2 = 1$$

Условие $x(2)$ выполнено.
